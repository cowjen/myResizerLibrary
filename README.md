# resizer-lib

## Features

myResizerLibrary enables change width and height of a custom HTML containers

## Installation

```html
<script type="text/javascript" src="resizer.js"></script>
```

## How to use

```html
<div id="myDiv"></div>
```

```js
(function () {
  myResizerLibrary.createResizer("myDiv");
})();
```
